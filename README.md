# 🔋 Energy Consumption Forecasting

This project predicts future energy consumption using historical data and machine learning. It includes a simple web interface built with Streamlit that allows users to interact with the model and visualize 24-hour forecasts.

## 🚀 Features

- Load and process energy consumption data
- Create lag and rolling statistical features
- Train a regression model (e.g., Linear Regression)
- Predict future energy consumption
- Interactive Streamlit dashboard for visualization
- Deployable on platforms like Streamlit Cloud or local server
